What is CQL

CQL is a schema definition and data query language.

Because it is built on the latest parsing algorithms it is able to read much like natural language while still being formally specified. For instance (a super trivial example):

Family Name is written as String(20);
Given Names is written as String(20);
Family is identified by its Name;
Person is identified by Family and Given Names where
    Person is of one Family,
    Family includes Person,
    Person has one Given Names;

This is just natural enough that you can collaborate directly with domain experts. Instead of translating their description of the model into SQL you can collaborate directly on the model.
